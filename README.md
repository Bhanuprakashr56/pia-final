# 🚀 Product Intern Assignment – 2026

A clean, responsive website built for the **Vocallabs.ai × Subspace.money** Product Intern Assignment (2026).

---

## 📌 About the Project

This website presents the full product teardown assignment for two of India's most interesting startups:

- **Vocallabs.ai** — AI voice agents automating business calls with human-like fluency
- **Subspace.money** — Subscribe to anything, delivered in minutes

The site covers:
- Company features & moats
- The task brief & product pillars
- Feedback structure (Observed → Problem → Ship instead)
- Evaluation criteria
- Next steps & submission deadline

## 🚀 How to Run

### Option 1 – Open Directly
Just double-click `index.html` — it opens in any browser instantly. No setup needed.

### Option 2 – VS Code Live Server
1. Open the folder in VS Code
2. Install the **Live Server** extension (by Ritwick Dey)
3. Right-click `index.html` → **Open with Live Server**
